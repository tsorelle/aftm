The Knockout View component provides a way to insert blocks that use the KnockoutJS data binding library and Tops/Peanut, a framework in PHP and TypeScript that supports AJAX remote procedure calls.  More detailed documentation is found in /application/docs/knockout-view.

View models and additional source files are found in the /application/mvvm directories
